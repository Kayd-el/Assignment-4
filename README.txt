This folder contains the python code for a password generator.
The user will be prompted to enter any word as well as an 8 digit number.
The number order will be randomized and attached to the end of the word entered by the user, thus creating a new password.

DockerHub:https://hub.docker.com/repository/docker/benjamin10303/assignment-4-image 
Image is ran with: docker run -it assignment-4-image
